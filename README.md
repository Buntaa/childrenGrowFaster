# Children Grow Faster Mod for Bannerlord

Welcome to the **Children Grow Faster** mod for Mount & Blade II: Bannerlord! This mod aims to enhance your gameplay experience by allowing children in the game to grow up faster.

## Features

- **Accelerated Growth**: Children in the game will grow up at an accelerated rate, allowing you to see them mature and become useful members of your clan more quickly.


## Installation

1. **Download the Mod**: Download the latest version of the mod from NexusMods website.
2. **Install through Vortex**: Click the vortex button if you are using Vortex as your mod launcher. This will automatically place the mod files where they need to be.

**Manual Installation**: Follow these steps to install the mod manually:

1. **Download the Mod**: Download the latest version of the mod from the NexusMods website.
2. **Extract the Files**: Extract the downloaded archive to a temporary location.
3. **Copy the Mod Files**: Copy the extracted files to the `Modules` folder in your Mount & Blade II: Bannerlord installation directory. The typical path is `C:\Program Files (x86)\Steam\steamapps\common\Mount & Blade II Bannerlord\Modules`.
4. **Enable the Mod**: Launch the game and go to the `Mods` tab in the launcher. Ensure that the `Children Grow Faster` mod is enabled.



## Configuration
To configure the mod, use the Mod Configuration Menu (MCM). Navigate to the MCM settings in the game to customize the options available for this mod.

## Mod Conflictions

If you are wondering if this mod has any conflicts with other mods, it does not. UNLESS that mod is messing with the `DailyTickEvent` method in the `CampaignEvents` class and is adding logic that affects `Heroes`.
